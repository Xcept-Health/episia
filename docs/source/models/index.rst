models Module
=============

Compartmental epidemic models for infectious disease dynamics.

This module provides a comprehensive suite of compartmental models
(SIR, SEIR, SEIRD) with a unified interface, parameter validation,
calibration tools, sensitivity analysis, and scenario comparison.
